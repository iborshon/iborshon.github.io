Personal academic and professional portfolio site for **Ishraque Zaman Borshon**, PhD Candidate in Aerospace & Mechanical Engineering at the University of Arizona.

---

## To add your profile photo

1. Add a photo file named `photo.jpg` to this repository root
2. Edit `index.html` — find the `profile-photo` div and replace the placeholder with:
   ```html
   <img src="photo.jpg" alt="Ishraque Zaman Borshon">
   ```

## To update your ORCID

Find `0000-0000-0000-0000` in `index.html` and replace with your real ORCID ID.

## To enable CV download

Add your CV as `cv_ishraque_zaman_borshon.pdf` to the repository root. The "Download CV" button in the hero will work automatically.

## Local preview

Open `index.html` in any browser. No server needed.

## Updating the site

Replace `index.html` and commit — GitHub Actions redeploys within ~60 seconds.
